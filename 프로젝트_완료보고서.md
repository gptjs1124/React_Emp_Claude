## 1. 프로젝트 개요

사내 직원(Employee)과 부서(Department) 데이터를 관리하는 싱글 페이지 웹 애플리케이션(SPA).  
기존 Node.js/Express 백엔드(포트 8080)와 REST API로 통신하며,  
브라우저에서 CRUD(조회·등록·수정·삭제) 작업을 수행할 수 있다.

| 분류 | 기술 | 적용 버전 |

## 3. 아키텍처

### 3.1 데이터 흐름

```
페이지 컴포넌트 (pages/)
  → Zustand 스토어 액션 (store/)
    → API 함수 (api/)
      → Axios 인스턴스 (api/axiosInstance.js)
        → 백엔드 http://localhost:8080/api
```

### 3.2 라우팅 구조

| URL | 컴포넌트 | 설명 |
|-----|---------|------|
| `/` | `HomePage` | 대시보드 (부서 수·직원 수 StatCard) |
| `/departments` | `DepartmentListPage` | 부서 목록 + 등록 버튼 |
| `/departments/new` | `DepartmentFormPage` | 부서 등록 폼 |
| `/departments/:id/edit` | `DepartmentFormPage` | 부서 수정 폼 |
| `/employees` | `EmployeeListPage` | 직원 목록 + 등록 버튼 |
| `/employees/new` | `EmployeeFormPage` | 직원 등록 폼 |
| `/employees/:id/edit` | `EmployeeFormPage` | 직원 수정 폼 |

### 3.3 소스 파일 구조

```
emp-dept-app/src/
├── api/
│   ├── axiosInstance.js      ← baseURL, timeout, headers 설정
│   ├── departmentApi.js      ← 부서 CRUD API 함수
│   └── employeeApi.js        ← 직원 CRUD API 함수
├── store/
│   ├── departmentStore.js    ← Zustand 부서 스토어
│   └── employeeStore.js      ← Zustand 직원 스토어
├── pages/
│   ├── HomePage.jsx
│   ├── DepartmentListPage.jsx
│   ├── DepartmentFormPage.jsx
│   ├── EmployeeListPage.jsx
│   └── EmployeeFormPage.jsx
├── components/
│   ├── layout/
│   │   ├── Layout.jsx        ← NavBar + Outlet 래퍼
│   │   └── NavBar.jsx        ← 상단 네비게이션
│   ├── department/
│   │   ├── DepartmentCard.jsx
│   │   └── DepartmentForm.jsx
│   └── employee/
│       ├── EmployeeCard.jsx
│       └── EmployeeForm.jsx
├── App.jsx                   ← createBrowserRouter 라우터 설정
└── main.jsx                  ← 진입점
```

## 6. 주요 기술 결정사항

| 항목 | 결정 내용 |
|------|---------|
| Tailwind v4 설치 방식 | `@tailwindcss/vite` 플러그인 사용, `tailwind.config.js` 불필요 |
| React Router v7 import | `'react-router'`에서 직접 import (Link, Outlet, useNavigate, useParams 등) |
| Zustand 배열 업데이트 | `map`, `filter`, 스프레드 등 불변(immutable) 방식 준수 |
| 직원 수정 데이터 조회 | GET-by-id API 대신 Zustand 스토어의 employees 배열에서 find |
| 뮤테이션 후 상태 갱신 | API 응답값으로 로컬 스토어 직접 갱신 (재조회 없음) |
